PRESENCE NOTIFICATIONS - REAL-TIME ROOM UPDATES
═══════════════════════════════════════════════════════════════════════════════

FEATURE OVERVIEW:
─────────────────────────────────────────────────────────────────────────────
Users now receive real-time notifications when other users join or leave rooms.

NOTIFICATIONS SENT:

1. When user joins a room (/join <room>):
   OLD room members receive: [SERVER] <user> left <old_room>
   NEW room members receive: [SERVER] <user> joined <new_room>

2. When user leaves a room (/leave):
   OLD room members receive: [SERVER] <user> left <old_room>
   GENERAL room members receive: [SERVER] <user> joined general

IMPROVEMENTS MADE:
─────────────────────────────────────────────────────────────────────────────
Enhanced /join command to:
  ✓ Notify the old room that user is leaving
  ✓ Notify the new room that user is joining
  ✓ Prevent duplicate room joins
  ✓ Track room transitions for debugging

CODE CHANGES:
─────────────────────────────────────────────────────────────────────────────
File: server.py (handle_chat_command function, /join section)

ADDED:
  1. Check if user already in target room (avoid duplicate joins)
  2. Broadcast "left" message to old room before switching
  3. Broadcast "joined" message to new room after switching
  4. Enhanced logging: "moved from X to Y"

BEFORE (/join):
    leave_room(client_socket, current_room)
    join_room(client_socket, new_room, username)
    confirm_msg = f"[SERVER] You joined room: {new_room}\n"
    client_socket.send(confirm_msg.encode('utf-8'))
    broadcast_msg = f"[SERVER] {username} joined {new_room}\n"
    broadcast_to_room(broadcast_msg, new_room, sender_socket=client_socket)

AFTER (/join):
    # Don't rejoin same room
    if current_room == new_room:
        msg = f"[SERVER] Already in {new_room}\n"
        client_socket.send(msg.encode('utf-8'))
        return None, current_room
    
    # Notify old room that user is leaving
    leave_room(client_socket, current_room)
    leave_msg = f"[SERVER] {username} left {current_room}\n"
    broadcast_to_room(leave_msg, current_room)
    
    # Join new room
    join_room(client_socket, new_room, username)
    
    # Send confirmation to user
    confirm_msg = f"[SERVER] You joined room: {new_room}\n"
    client_socket.send(confirm_msg.encode('utf-8'))
    
    # Notify new room that user joined
    broadcast_msg = f"[SERVER] {username} joined {new_room}\n"
    broadcast_to_room(broadcast_msg, new_room, sender_socket=client_socket)
