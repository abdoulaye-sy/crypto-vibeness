PRIVATE MESSAGING (DM) - QUICK REFERENCE
════════════════════════════════════════════════════════════════════════════════

COMMAND SYNTAX
──────────────────────────────────────────────────────────────────────────────
  /msg <username> <message>

EXAMPLES
──────────────────────────────────────────────────────────────────────────────
  /msg alice hello there!
  /msg bob hey, how are you?
  /msg vincent salut bro!

RESPONSES
──────────────────────────────────────────────────────────────────────────────

If message sent successfully:
  [DM → alice]: hello there!

If message received:
  💬 PRIVATE MESSAGE 💬
  [DM from bob]: hey there!

If user not found:
  [ERROR] User 'xxx' not found

If user offline:
  [ERROR] User 'xxx' is not online

If message empty:
  [ERROR] Message cannot be empty

If usage wrong:
  [ERROR] Usage: /msg <username> <message>

REQUIREMENTS
──────────────────────────────────────────────────────────────────────────────
✓ Both users must be registered
✓ Recipient must be currently online/connected
✓ Message cannot be empty
✓ Username must exist in database
