WHAT WAS FIXED:
───────────────

FILE: client.py (Lines 190-199)

BEFORE:
    message = parts[1].strip() if len(parts) > 1 else ""
    message = decrypt(message)

ISSUE: 
    The .strip() method removes whitespace, including control 
    characters like \x0b (vertical tab) that can appear in encrypted 
    content. This caused some messages to be truncated.
    
    Example:
    - Encrypted "123456789" → "RZR@^SN\t\x0b" (9 chars)
    - After strip() → "RZR@^SN" (7 chars)
    - Decrypted as → "1234567" ❌ WRONG

AFTER:
    message = parts[1] if len(parts) > 1 else ""
    message = message.lstrip() if message.startswith(' ') else message
    if '\n' in message:
        message = message.split('\n')[0]
    message = decrypt(message)

SOLUTION:
    - Use lstrip() with condition to only remove leading spaces
    - Split on actual newline to separate from MSG_ID
    - Preserve all encrypted binary content
    - Then decrypt the full encrypted message
    
    Example:
    - Encrypted "123456789" → "RZR@^SN\t\x0b" (9 chars, intact)
    - After split → "RZR@^SN\t\x0b" (9 chars, preserved)
    - Decrypted as → "123456789" ✅ CORRECT
