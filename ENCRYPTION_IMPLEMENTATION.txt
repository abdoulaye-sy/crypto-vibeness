IMPLEMENTATION DETAILS
=======================

1. ENCRYPTION METHOD: XOR Cipher
   - Simple, deterministic symmetric encryption
   - Same operation for encrypt/decrypt (XOR is self-inverse)
   - Repeating key pattern for longer messages

2. SHARED KEY:
   ENCRYPTION_KEY = 'chatkey123'
   - Both server and client use same key
   - 10 characters long
   - Repeats for messages longer than key

3. FUNCTIONS ADDED:

   Server (server.py):
   
   def encrypt(message):
   - Takes plaintext message
   - Returns XOR-encrypted bytes string
   - Uses ENCRYPTION_KEY for cipher
   
   def decrypt(message):
   - Takes encrypted message
   - Returns decrypted plaintext
   - XOR is symmetric, so same as encrypt()
   
   def should_encrypt_message(message):
   - Determines if message needs encryption
   - Returns False for commands (/)
   - Returns False for [SERVER] messages
   - Returns True for regular messages
   
   Client (client.py):
   
   def encrypt(message)
   def decrypt(message)
   - Identical to server versions
   - Used to decrypt received messages

ENCRYPTION LOGIC
=================

XOR Process:
For each character at position i in message:
  1. Get key_char = ENCRYPTION_KEY[i % len(ENCRYPTION_KEY)]
  2. Calculate: encrypted_char = ord(char) XOR ord(key_char)
  3. Convert result to character

Example:
  Message: "hello"
  Key:     "chatkey123" (repeats as "chatch")
  
  'h' XOR 'c' = chr(104 ^ 99) = chr(7)
  'e' XOR 'h' = chr(101 ^ 104) = chr(13)
  'l' XOR 'a' = chr(108 ^ 97) = chr(13)
  'l' XOR 't' = chr(108 ^ 116) = chr(24)
  'o' XOR 'k' = chr(111 ^ 107) = chr(4)
  
  Encrypted: chr(7) + chr(13) + chr(13) + chr(24) + chr(4)

Decryption:
  chr(7) XOR 'c' = chr(7 ^ 99) = chr(104) = 'h'
  chr(13) XOR 'h' = chr(13 ^ 104) = chr(101) = 'e'
  etc... → "hello"

INTEGRATION POINTS
===================

Server-Side Encryption:

1. Room Messages (handle_client):
   Before: msg_text = f"{username}: {message}\nMSG_ID:{msg_id}\n"
   After:  
   - encrypted_msg = encrypt(message) if should_encrypt_message(message)
   - msg_text = f"{username}: {encrypted_msg}\nMSG_ID:{msg_id}\n"

2. DM Messages (handle_dm_command):
   Before: dm_msg = f"[DM from {sender_username}]: {message}\n"
   After:
   - encrypted_dm = encrypt(message)
   - dm_msg = f"[DM from {sender_username}]: {encrypted_dm}\n"

Client-Side Decryption:

1. Regular Messages (parse_and_display_message):
   Before: message = parts[1].strip()
   After:
   - message = parts[1].strip()
   - message = decrypt(message)

2. DM Messages (parse_and_display_message):
   Before: Display [DM from sender]: encrypted_content
   After:
   - Extract encrypted content
   - decrypted_msg = decrypt(encrypted_content)
   - Display [DM from sender]: decrypted_msg

MESSAGE FLOW
============

Room Message:
  User sends: "hello"
  → Server receives: "hello"
  → Encrypts: encrypt("hello") = '\x0b\r\r\x18\x04'
  → Broadcasts: "alice: \x0b\r\r\x18\x04\nMSG_ID:1\n"
  → Client receives encrypted message
  → Decrypts: decrypt('\x0b\r\r\x18\x04') = "hello"
  → User sees: "alice: hello"

DM Message:
  User sends: "/msg bob secret"
  → Command parsed
  → Content extracted: "secret"
  → Encrypts: encrypt("secret") = '\x16\x1cKP\x05G'
  → Sends DM: "[DM from alice]: \x16\x1cKP\x05G\n"
  → Bob receives encrypted
  → Decrypts: decrypt('\x16\x1cKP\x05G') = "secret"
  → Bob sees: "[DM from alice]: secret"
